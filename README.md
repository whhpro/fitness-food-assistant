# 健身饮食助手

一个手机优先的训练、饮食、有氧和游泳记录 WebApp。

## 功能

- 记录常用器械动作：高位推胸、高位下拉、划船、引体、蝴蝶机夹胸、腿屈伸、深蹲等
- 记录力量训练重量、组数、次数、时长和疲劳感
- 记录爬坡有氧：时长、坡度、速度和强度
- 记录游泳
- 记录早餐、午餐、晚餐、加餐、夜宵，并做本地热量粗估
- 保存名字、头像、身高体重和训练目标
- 根据最近训练和本周分布给出当天建议

## 数据说明

当前版本的数据保存在浏览器本地，不会上传服务器。

换手机、换浏览器、清理浏览器数据后，本地记录可能丢失。后续可以继续加“导出/导入数据”和云同步。

## 本地运行

```bash
python3 -m http.server 4174
```

然后打开：

```text
http://localhost:4174
```
